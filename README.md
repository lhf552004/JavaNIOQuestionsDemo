# Introduction
This is for Java NIO
Java NIO（新IO）是Java的替代IO API，意味着可以替代标准 Java IO和Java Networking API。

Java NIO提供了与传统IO API不同的IO编程模型。

注意：有时NIO被称为是非阻塞IO。但是，这并不是NIO最初的意思。此外，NIO API的某些部分实际上正在阻塞-例如文件API-因此，“非阻塞”标签会引起误解。
